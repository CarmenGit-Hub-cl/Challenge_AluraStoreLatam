# 📊 Alura Store Latam – Análisis de Ventas y Rendimiento de Tiendas

## 📌 Descripción del proyecto

Este proyecto tiene como objetivo analizar el rendimiento de las cuatro tiendas de **Alura Store Latam** para ayudar al **señor Juan** a tomar una decisión estratégica sobre cuál tienda debería vender para invertir en un nuevo negocio.

Para ello, se realizó un análisis de datos utilizando **Python y Google Colab**, evaluando distintos indicadores de desempeño de cada tienda.

El análisis se basa en cinco aspectos principales:

- Facturación total de cada tienda
- Categorías de productos más populares
- Calificación promedio de los clientes
- Productos más y menos vendidos
- Costo promedio de envío

A partir de estos indicadores se determina cuál tienda presenta **menor rendimiento**.

---

# 🎯 Objetivo del análisis

El objetivo principal del proyecto es **identificar la tienda con menor desempeño comercial** para recomendar su venta y permitir al señor Juan **reinvertir en un nuevo negocio con mayor potencial**.

El análisis busca responder preguntas como:

- ¿Cuál tienda genera más ingresos?
- ¿Qué categorías de productos son las más populares?
- ¿Cómo evalúan los clientes a cada tienda?
- ¿Qué productos tienen mayor y menor rotación?
- ¿Cuál tienda tiene los mayores costos de envío?

# 📊 Análisis realizados

## 1️⃣ Facturación total por tienda

Se calculó la suma total de ventas de cada tienda para identificar cuál genera mayores ingresos.

Este indicador permite comparar el **desempeño financiero general** de cada tienda.

---

## 2️⃣ Ventas por categoría

Se analizó cuáles son las **categorías de productos más vendidas** en cada tienda.

Esto permite identificar:

- Preferencias de los clientes
- Diferencias en la demanda entre tiendas

---

## 3️⃣ Calificación promedio de clientes

Se calculó el promedio de **evaluaciones de clientes** para cada tienda.

Este indicador refleja el **nivel de satisfacción del cliente**.

---

## 4️⃣ Productos más y menos vendidos

Se identificaron:

- Los **productos con mayor volumen de ventas**
- Los **productos con menor rotación**

Esto permite entender mejor el comportamiento de compra.

---

## 5️⃣ Costo promedio de envío

Se calculó el promedio del **costo de envío** de cada tienda.

Este indicador es importante porque **costos de envío altos pueden afectar la competitividad de una tienda**.

---

# 🧾 Conclusión del análisis

Luego de analizar los cinco indicadores principales, se observó que **la Tienda 4 presenta el menor desempeño general** en comparación con las otras tiendas.

En particular:

- Presenta **menor facturación total**
- Tiene **menor volumen de ventas**
- Su rendimiento global es inferior al de las otras tiendas

Por esta razón, se recomienda al **señor Juan considerar la venta de la Tienda 4**, permitiéndole liberar recursos para invertir en un nuevo negocio con mayor potencial de crecimiento.
